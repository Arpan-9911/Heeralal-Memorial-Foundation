# Heeralal Memorial Foundation 🌿

A modern full-stack web application built to support and manage activities for the Heeralal Memorial Foundation. This platform aims to provide a seamless experience for users to explore, connect, and engage with the foundation.

---

## 🚀 Tech Stack

### Frontend

* React (Vite)
* Tailwind CSS
* React Router DOM
* Zustand / Redux Toolkit
* React Query
* Axios
* Framer Motion
* Sonner (Toast Notifications)

### Backend (Planned / Integrated)

* Node.js
* Express.js
* MongoDB

---

## 📂 Project Structure

```
src/
│
├── assets/            # Images, icons, static files
├── components/
│   ├── common/        # Reusable UI components
│   └── layout/        # Navbar, Footer, Sidebar
├── layouts/           # Layout wrappers
├── pages/             # Application pages
├── services/          # API calls
├── store/             # State management
│
├── App.jsx
├── main.jsx
└── index.css
```

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/Arpan-9911/Heeralal-Memorial-Foundation.git
cd Heeralal-Memorial-Foundation
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```
